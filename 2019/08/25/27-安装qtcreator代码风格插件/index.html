安装qtcreator代码风格插件

  1. 在Help→About plugins安装代码格式的beautifer插件

  2. 重启qtcreator

  3. Tool→Option 选择Beautifier

  4. 配置

  5. 配置内容

–style=attach
–indent=spaces=4
–align-pointer=name
–align-reference=name
–convert-tabs
–attach-namespaces
–max-instatement-indent=120
–pad-header
–pad-oper
6. 安装 sudo apt-get install astyle 即可
