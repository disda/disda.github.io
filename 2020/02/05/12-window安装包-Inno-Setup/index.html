12-window安装包-Inno_Setup

  1. 下载安装innosetup-6.0.3 https://www.jrsoftware.org/isdl.php
    build youduqt release版本
    在youduqt.exe文件下cmd执行windeployqt.exe youduqt.exe，这样会将相应的依赖库放到与youduqt.exe相应的目录下。

  2. 新建安装包脚本youduqt.iss 添加安装语言添加图标
    默认没有中文简体和繁体语言包http://www.jrsoftware.org/files/istrans/
    添加卸载清空文件夹 代码
    [UninstallDelete]
    Type: files; Name: “{app}\ca-bundle.crt”
    Type: filesandordirs; Name: “{app}\bearer”
    Type: filesandordirs; Name: “{app}\iconengines”
    Type: filesandordirs; Name: “{app}\imageformats”
    Type: filesandordirs; Name: “{app}\platforms”
    Type: filesandordirs; Name: “{app}\sqldrivers”
    Type: filesandordirs; Name: “{app}\translations”
    Type: dirifempty; Name: “{app}”

  3. 添加安装前卸载前退出执行程序
    [Code]
    var ErrorCode: Integer;
    function InitializeSetup(): Boolean;
    begin
    ShellExec(‘open’, ExpandConstant(‘{cmd}’), ‘/c taskkill /f /t /im youduqt.exe’, ‘’, SW_HIDE, ewNoWait, ErrorCode);
    Result:= true;
    end;

function InitializeUninstall(): Boolean;
begin
ShellExec(‘open’, ExpandConstant(‘{cmd}’), ‘/c taskkill /f /t /im youduqt.exe’, ‘’, SW_HIDE, ewNoWait, ErrorCode);
Result:= true;
end;

运行生成相应的安装包
