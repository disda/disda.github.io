  1. 1. 编译
  2. 2. 使用

编译

  1. 下载google-breakpad 在25上有一份(/dailybuild/linux_qt/package/breakpad.zip)

-2.svn checkout http://google-breakpad.googlecode.com/svn/trunk/ google-breakpad-read-only
3. ./configure
make
sudo make install

使用

  1. 找到代码库提交的symbolstore.py文件（https://dxr.mozilla.org/mozilla-central/source/toolkit/crashreporter/tools/symbolstore.py?raw=1）
  2. 执行python /path/to/script/symbolstore.py /usr/local/bin/dump_syms ./symbol-storage ./Application
  3. dump_syms为一个可执行文件，./symbol-storage 为存放的路径，./Application为应用路径
  4. 将在./symbol-storage路径下生成文件夹Application/uuid同事copy生成的.sym文件到该文件夹
